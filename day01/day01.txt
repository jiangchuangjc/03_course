1.shell指令
  Linux中文件夹也叫做目录
  1) pwd,cd,ls
    pwd:print working directory
        打印当前所在的路径
    cd:change directory
       切换工作目录
       cd a 切换到当前目录下的a目录中去
       cd .. 切换到上一级目录
       cd a/b/c/d
       cd ~ :表示当前登录用户的主目录   
    ls:list
       显示当前目录中的所有非隐藏文件或目录
       紫色(后缀/):表示目录/文件夹
       白色:表示文件
       绿色(后缀*):表示可执行文件
       .lnk:超链接,快捷方式
